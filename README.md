# Mandora Belle Frontend

Frontend website e-commerce fashion Mandora Belle menggunakan Vue.js + TypeScript.

## Features

- Product Catalog
- Product Detail
- Shopping Cart
- Checkout System
- Payment Integration
- Order History
- Responsive UI

## Technologies

- Vue.js
- TypeScript
- Vite
- Pinia
- Vue Router
- Tailwind CSS

## Installation

1. Clone repository

2. Install dependency

```bash
npm install

3. Run frontend
npm run dev

4. Open browser
http://localhost:5173
